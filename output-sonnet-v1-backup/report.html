Methodology & Data Quality

Generated by verum-extract v0.1.0, an open-source LLM-powered extraction pipeline.

  • Source: 2,162 posts + comments from r/Peptides scraped March 30, 2026 (public data)
  • Model: Claude Sonnet (claude-sonnet-4-6) via Anthropic Batch API
  • Extraction: Each post + its comments sent as individual API call; separate report per person
  • Normalization: 72 built-in compounds + 695 auto-discovered via alias/fuzzy matching
  • Confidence tiers: T3 = biomarkers/labs, T4 = detailed protocol, T5 = vague mention
  • Side effect note: This run used 14 side effect categories. v0.2 expands to 27 categories (adding GI, skin reaction, cardiac, joint pain, cognitive, allergic, anxiety, hair, vision, hormonal, respiratory, muscle cramp, weight change) which should reduce "other" from 33% to under 10%
  • Limitations: Self-reported, unverified data. 56% avg LLM confidence. Not medical advice.
